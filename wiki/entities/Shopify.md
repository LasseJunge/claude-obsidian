# Shopify

Ecommerce platform company; relevant here as the publisher of [[Under the River — Shopify Agent Platform]], describing its internal AI-agent infrastructure.

- **River** — Slack-native AI coding agent; coauthors ~1 in 8 merged PRs company-wide.
- **[[Aquifer Agent Platform]]** — internal platform running the agents (brain/hands split, durable sessions).
- **World** — single monorepo (code + skills + runbooks) enabling agent navigation.
- **[[Nix]]** — reproducible environments across dev/CI/prod.

Note: a direct ecommerce-platform peer of [[ePages]]; their agent-infrastructure playbook is a useful reference for any ePages internal-agent ambitions.
