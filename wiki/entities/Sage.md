# Sage

Payment/accounting provider integrated with ePages via a connector. As of the [[meeting-2026-06-15-with-Chris|2026-06-15 meeting]], Sage is rolling out a **new connector (HTK)**.

## Context in ePages

- A **new Sage Connector** ("HTK") replaces the previous integration.
- Roughly **350 Base shops** must **update to remain using Sage** — shops that don't update lose the integration.
- Feature usage should be tracked (which Sage features shops actually use) against **Kristof's sheet** — confirm whether that sheet is current.
- Tracking approach mirrors the [[ePages Spreedly Migration]] / [[Spreedly]] model.
