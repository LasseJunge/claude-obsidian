# Local LLM Hub Plugin

## Kernfunktionen

**Workflow Automation**: Aufgaben in natürlicher Sprache beschreiben → KI baut ausführbaren Workflow. Kein YAML nötig. Visueller Node-Editor mit 23 Node-Typen.

**RAG (Retrieval-Augmented Generation)**: Lokale semantische Suche über den gesamten Vault via Embeddings. Alles wird lokal berechnet und gespeichert — keine externen Dienste.

**Privacy-First**: Alle Verarbeitung auf dem eigenen Rechner. Unterstützt Ollama, LM Studio, vLLM, AnythingLLM.

## Lernen durch iterative Verbesserung

Das System "lernt" nicht im ML-Sinne — es verfeinert sich iterativ: Workflows können per Feedback regeneriert werden; Edit-History erlaubt Rollback und Diff-Ansicht aller KI-Änderungen.
