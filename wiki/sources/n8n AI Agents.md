# n8n AI Agents

n8n is a no-code/low-code visual workflow platform with native AI-agent support.

- **Autonomy:** agents decide and act via tool access; supports multi-agent teams (research/writing/QA).
- **Scraping:** ships templates "AI agent that can scrape webpages" and "Scrape and summarize webpages with AI."
- **Connectivity:** 500+ pre-built nodes + HTTP Request node for any API.
- **Deep research:** "Deep Research Agents" mine large data with API access + memory for structured takeaways (i.e. multi-source comparison).
- **Models:** OpenAI, Anthropic Claude, any LLM/vector store.
- **Accessibility:** built for technical and non-technical users; drag-and-drop; open-source + self-hostable.

Best-fit orchestrator for "scrape listings → compare to rent index → report on a schedule."
