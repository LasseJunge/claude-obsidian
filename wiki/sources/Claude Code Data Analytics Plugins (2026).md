# Claude Code Data Analytics Plugins (2026)

## Marketplace plugins (external)

| Plugin | What it does |
|--------|-------------|
| `analytics-reporter` | Transforms raw metrics into actionable reports; insight generation + recommendations |
| `data-scientist` | SQL/BigQuery specialist agent; proactive on analytical tasks |
| `experiment-tracker` | A/B test monitoring, success metrics, feature flag management |
| `feedback-synthesizer` | Converts multi-source user feedback into product insights |
| `trend-researcher` | Market opportunity identification, viral content analysis |

## claude-obsidian built-in data skills (already in this vault)

The `data:` skill group ships with this plugin:

| Skill | Use |
|-------|-----|
| `data:analyze` | Statistical analysis on any dataset |
| `data:build-dashboard` | Scaffold a full React+Recharts dashboard |
| `data:create-viz` | Single chart/visualization |
| `data:explore-data` | EDA: distributions, correlations, outliers |
| `data:sql-queries` | Write and optimize SQL |
| `data:statistical-analysis` | Hypothesis testing, regression |
| `data:validate-data` | Catch inconsistencies, nulls, type errors |
| `data:write-query` | Targeted query authoring |

## AI-Analyst toolkit (GitHub: ai-analyst-lab/ai-analyst)

18-agent pipeline: business question framing → data exploration → root cause → narrative → validated slide deck with speaker notes. Best for ad-hoc investigative analytics.
